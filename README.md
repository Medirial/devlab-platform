# DevLab Platform

## Objectif
Plateforme web moderne pour quiz techniques, roadmaps de développement, suivi de progression, et apprentissage cloud.

## Architecture

- **frontend/** : Application Next.js (React, TypeScript)
- **backend/** : API Node.js + Express
- **docker/** : Fichiers de containerisation (à venir)
- **docs/** : Documentation technique et fonctionnelle

## Démarrage rapide

- Frontend :
  - `cd frontend && npm run dev`
- Backend :
  - `cd backend && node index.js`

## À venir
- Dockerisation
- Déploiement AWS (EC2)
- CI/CD
- Authentification, quiz, progression, etc.
